# 🌐 05 Sitio Web Oficial

El espacio designado para alojar el código fuente, configuración y recursos (assets web) que compondrán el front-end oficial del colectivo en internet, enlazando directamente al CRM y al portafolio B2B.

## 📁 Tipos de contenidos alojados aquí
- En adelante, esta carpeta alojará el archivo principal `index.html` e `index.css` de un sitio tipo *Landing Page* moderno. Si el colectivo necesita funcionalidades complejas como formularios interactivos de cotización automática y tableros para clientes empresariales, el sitio mutará a un ecosistema o un framework de componentes reactivos.
- **Copywriting SEO:** Los textos optimizados para Google acerca de servicios de "Comunicación Científica Transmedia", "Agencia Biotecnología Valle del Cauca". 

> **💡 Regla del Colectivo:** Por ahora y al ser "Lean", el objetivo 1 del Sitio Web NO es vender al por mayor, sino fungir como "Validador de Confianza". Si mandas un prospecto vía Linkedin en frío, él abrirá la web oficial solo para validar que eres un experto sólido y bien diseñado. ¡El diseño debe ser impecable (Glassmorphism, dark modes, tipografía de alta fidelidad)!
