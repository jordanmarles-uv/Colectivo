# ⚖️ 01 Administración y Legales

Este directorio contiene los cimientos burocráticos, financieros y organizativos del colectivo para operar conforme a la ley colombiana, manteniendo el modelo "Lean" y "Persona Natural".

## 📁 Tipos de contenidos alojados aquí
1. **Contratos y Licenciamientos:** Borradores de prestación de servicios, acuerdos de confidencialidad (NDA) para ciencias duras, y las cláusulas base de co-autoría moral avaladas por la DNDA.
2. **Facturación y RUT:** Registros, extractos de declaración ante la DIAN, controles de validación para no exceder las 3.500 UVT anuales obligadas para el régimen de No Responsable de IVA.
3. **Identidad Institucional:** Cartas de presentación formales del colectivo, NIT (si aplica a futuro), y certificados de representación legal o credenciales comerciales.

> **💡 Regla del Colectivo:** Ningún proyecto debe iniciar hasta que exista un contrato o acuerdo pre-firmado guardado en esta carpeta que blinde el aspecto de coautoría y los alcances del entregable transmedia.
