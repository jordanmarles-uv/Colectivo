# 🚀 04 Proyectos Activos (Caja de Arena)

Esta es la carpeta operativa diaria de ejecución. Todos los trabajos solicitados por clínicas, universidades, laboratorios o minciencias deben vivir aquí estructurados uniformemente para no generar caos con los archivos perdidos.

## 📁 La regla de Oro Operativa: El Template
Adentro de esta carpeta encontrarás la subcarpeta maestra: **`_Template_Nuevo_Proyecto`**.
NUNCA modifiques directamente el Template. Cada vez que inicie un proyecto B2B, copia la carpeta entera `_Template_Nuevo_Proyecto` y cámbiale el nombre por algo identificable ej. `2025_04_Pitch_ValleyCare_HealthCorp`.

## 📁 ¿Qué hay dentro de la carpeta de un Proyecto Activo?
1. **`01_Assets`**: Carpeta donde lanzamos todo el material provisto por el cliente sin procesar (Logos vectoriales del cliente, fotos institucionales en alta, videos crudos).
2. **`02_Research_Y_Data`**: Aquí se ubican los Papers PDF originales de la investigación, Excels crudos a graficar, y el `01_Brief_Transmedia.md` procesado por el Agente Fact-Checker para "traducir" la jerga a idioma mortal.
3. **`03_Entregables`**: Los archivos finales e iteraciones entregadas (Exports, zips, micrositios empaquetados, guiones finales de copy).

> **💡 Metodología Lean:** Cuando el proyecto finalice comercialmente su fase de garantías, deberá subirse a una nube de respaldo (Google Drive / S3) y borrarse del entorno local de "Proyectos Activos" tras haber extraído las piezas valiosas para llevarlas a la carpeta `03_Portafolio_Assets/Casos_de_Exito`.
