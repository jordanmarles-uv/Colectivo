# 🎨 03 Portafolio y Assets Demos

Esta carpeta almacena todo el material audiovisual pre-diseñado para "vender visualmente" los servicios transmedia del colectivo. Dado que a las Startups (ej HealthTech) les cuesta imaginar un "Scrollytelling", tu trabajo aquí es tener Demos simulados y Casos Clínicos listos para mostrar.

## 📁 Tipos de contenidos alojados aquí
1. **Demos de Pitch Decks (One-Pagers):** PDFs y mockups del "Antes/Después" de presentaciones aburridas convertidas en obras modulares de 10 láminas que los Venture Capitals (VC) aprecian.
2. **Demos de Scrollytelling:** Extractos animados, GIFs y wireframes de cómo luce la interactividad para visualizar gráficas en la web.
3. **Casos de Éxito (Estudios de Caso):** Archivos donde documentamos victorias comerciales reales: *El reto técnico -> Nuestra solución gráfica -> El Crecimiento o levantamiento de capital obtenido*.
4. **Brand Assets Propios:** Logos oficiales del colectivo, paletas de colores maestras, tipografías y banners para redes sociales operacionales.

> **💡 Regla del Colectivo:** Nunca envíes a un cliente potencial a imaginar cómo se vería su proyecto; envíale un link directo a uno de los *Demos* que se almacenen en los subdirectorios de esta carpeta.
